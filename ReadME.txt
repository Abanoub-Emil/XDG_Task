ReadME File 

This folder contains the requested ANDROID challenge

The Android Studio version is 3.2.1 supporting all kinds of devices and screens 
you will have to clone the project then build and run 

The project architecture is MVP

I used a helper class which was taken from an opensource code so I can have the control to manipulate the speed of the marquee displayed by a simple equation

also used Retrofit and simpleXML to request BBC news and parse the XML to retrieve news description from tags
